# Es Tu Dia Jugant

Web educativa en català per a nens i nenes, pensada per aprendre jugant. Ideal per a tauletes i mòbils.

## Característiques
- Menú principal amb grans botons i navegació intuïtiva
- Sistema d'autenticació amb usuari i contrasenya
- Pista de contrasenya amb "toast" visual
- Mostra/oculta la contrasenya amb icona d'ull
- Disseny responsive, colors vius i tipografia clara
- Favicon i manifest per a PWA
- Llicència Creative Commons CC BY 4.0

## Instal·lació
1. Clona el repositori
2. Puja la carpeta a GitHub Pages o a un servidor web
3. Edita `data/users.json` per afegir usuaris
